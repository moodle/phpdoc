static

Unpack filteroptions passed in a request's filter param if required.

Filteroptions are passed via AJAX as an array of {name:, value:} pairs for compatibility with external functions.

Parameters
array$filtersList of filters, each optionally including an array of filteroptions.
Return values
arrayThe input array, with filteroptions unpacked from [{name:, value:}, ...] to [name => value, ...].
