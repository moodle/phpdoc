Returns a cell's PHP Date value, associated to the given timestamp.

NOTE: The timestamp is a float representing the number of days since the base Excel date: Dec 30th 1899, 1900 or Jan 1st, 1904, depending on the Workbook setting. NOTE: The timestamp can also represent a time, if it is a value between 0 and 1.
