Detailed Description

This class is used by cURL class, use case:

$CFG->repositorycacheexpire = 120; $CFG->curlcache = 120;

$c = new curl(array('cache'=>true), 'module_cache'=>'repository'); $ret = $c->get('http://www.google.com');
