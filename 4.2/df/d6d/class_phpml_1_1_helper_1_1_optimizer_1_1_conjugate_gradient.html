◆ getAlpha()

Phpml\Helper\Optimizer\ConjugateGradient::getAlpha ( array $d)
protected

Calculates alpha that minimizes the function f(θ + α.d) by performing a line search that does not rely upon the derivation.

There are several alternatives for this function. For now, we prefer a method inspired from the bisection method for its simplicity. This algorithm attempts to find an optimum alpha value between 0.0001 and 0.01

Algorithm as follows: a) Probe a small alpha (0.0001) and calculate cost function b) Probe a larger alpha (0.01) and calculate cost function b-1) If cost function decreases, continue enlarging alpha b-2) If cost function increases, take the midpoint and try again

Phpml\Helper\Optimizer\ConjugateGradient::getBeta ( array $newTheta)

Calculates new beta (β) for given set of solutions by using Fletcher–Reeves method.

β = ||f(x(k+1))||² ∕ ||f(x(k))||²

See: R. Fletcher and C. M. Reeves, "Function minimization by conjugate gradients", Comput. J. 7 (1964), 149–154.
