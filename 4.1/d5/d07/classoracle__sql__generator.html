Detailed Description

This class generate SQL code to be used against Oracle It extends XMLDBgenerator so everything can be overridden as needed to generate correct SQL.

◆ getAlterFieldSQL()

oracle_sql_generator::getAlterFieldSQL ( $xmldb_table,
$xmldb_field,
$skip_type_clause = NULL,
$skip_default_clause = NULL,
$skip_notnull_clause = NULL )

Given one xmldb_table and one xmldb_field, return the SQL statements needed to alter the field in the table.

Oracle has some severe limits:

  • clob and blob fields doesn't allow type to be specified
  • error is dropped if the null/not null clause is specified and hasn't changed
  • changes in precision/decimals of numeric fields drop an ORA-1440 error
Parameters

xmldb_field$xmldb_fieldThe instance of xmldb_field to create the SQL from.
string$skip_type_clauseThe type clause on alter columns, NULL by default.
string$skip_default_clauseThe default clause on alter columns, NULL by default.
string$skip_notnull_clauseThe null/notnull clause on alter columns, NULL by default.
Return values
stringThe field altering SQL statement.
